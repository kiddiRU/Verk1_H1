# Project folder

## Coding standard

### Python standard
- Using Pep 8 
- snake case 
- Author of code should write their name and purpose of code in the first lines in a file
- Type hinting. 
- Branch when creating a new feature

### Project standard
- CRUD naming conventions for 3-tiered functions
- UILayer: Add, show, edit, Remove
- LogicLayer: Create, get, change, Delete
- DataLayer: Store, load, Update, Discard

### Header of file
Author: Full Name
Date: YYYY-MM-DD

Description of purpose
